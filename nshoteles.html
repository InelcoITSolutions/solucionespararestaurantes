National Soft ® Hoteles es la solución ideal para la administración de tu hotel o motel que se adapta a tus necesidades

Administra tu hotel o motel de manera eficiente, eliminando el trabajo manual y mejorando su control

NS Hoteles es un sistema que cumple con las necesidades que tu hotel requiere. El programa realiza de forma eficaz y eficiente los procesos comunes (entrada y salida de huéspedes, movimientos, reservaciones y cobro), ofreciendo así el control adecuado de tu hotel o motel.

NS Hoteles cuenta con niveles de acceso al sistema. Se puede restringir el acceso a los diferentes módulos y funcionalidades mediante contraseñas y un módulo especial de huella digital. Además te permite dar el mantenimiento necesario a tu sistema para tener siempre un óptimo funcionamiento y evitar problemas.

Su facilidad de uso convierte a NS Hoteles en una opción indispensable para tu negocio.

    Ticket de consumo

    Impresión de ticket de compra de consumo al pagar

    Modifique folios (cuentas) históricos (cargos, abonos, edición y cancelación) para corregir aclaraciones, requiere permiso del usuario

Requerimientos

  • Procesador Intel® Core i3 2.0 Ghz o superior
  • Memoria RAM 4 GB
  • Disco Duro 80GB disponibles
  • Monitor 1024 x 768
  • Sistema Operativo
  • Windows® 7 Professional SP2 ó Ultimate, Windows 8.1 Pro o superior.
  • Requisitos adicionales del Servidor: SQL Server 2008 (mínimo), servidores y PC’s en español
  • Unidad de DVD ROM, USB 2.0, Puerto de Red Ethernet, Conexión a Internet para registro de licencia.
